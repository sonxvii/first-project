Список терминов

Краткий список основных терминов приведен ниже:

Атрибут
Cпециальные слова, которые управляют поведением HTML-элемента. Они добавляют дополнительную функциональность, либо меняют поведение элемента по умолчанию. Атрибуты элемента выражаются внутри начального тега элемента. Атрибут имеет имя и значение
Верстка
Это структурированное сочетание изображений, заголовков, подзаголовков, таблиц, инфографик, текста и других элементов на странице c помощью языка разметки HTML и языка описания внешнего вида страницы CSS
Тег, теги (англ. tag — именованная метка)
Элемент языка разметки гипертекста. Текст, содержащийся между начальным и конечным тегом, отображается и размещается в соответствии со свойствами, указанными в начальном теге.

Например, тег исправления <Del> позволяет зачеркнуть текст: Эта часть текста зачёркнута, а эта – нет.

Сочетания клавиш
При помощи сочетания клавиш можно ускорить процесс работы с кодом. Например, сочетание <Ctrl> + <C> позволяет копировать выделенный элемент, а <Ctrl> + <V> – вставить.
Форматирование кода
Форматирование кода не влияет на его функицонирование, однако правильное форматирование и комментирование делает программные исходники более понятными для человека. Отформатировать код можно с помощью сочетания клавиш <Shift> + <Alt> + <F>

Например, такое отображение кода вызывает затруднения в его понимании:
<!DOCTYPE HTML> <html> <body> <script> alert( 'Hello, world!' ); </script> </body> </html>
